# Company Overview — NovaTech Solutions

## About Us

NovaTech Solutions is a B2B SaaS company founded in 2018, headquartered in Warsaw, Poland.
We build cloud-based tools for supply chain management and logistics automation.
Our platform is used by over 300 enterprise clients across Europe and North America.

## Mission

Simplify complex logistics operations so that companies can focus on growth, not on paperwork.

## Core Values

- **Transparency** — we share context openly, both internally and with clients.
- **Ownership** — every team member takes responsibility for outcomes, not just tasks.
- **Continuous improvement** — we ship fast, learn from feedback, and iterate.
- **Respect** — we treat each other and our clients as partners.

## Products

### LogiCore Platform
Our flagship SaaS product. Provides real-time visibility into supply chains, automated
order routing, and predictive inventory management. Deployed on AWS, available as
multi-tenant SaaS or single-tenant private cloud.

### NovaTech API
A REST API layer that allows clients to integrate LogiCore data into their own ERP systems.
Supports webhooks, OAuth 2.0, and rate-limited public access.
